# 아차차! 피하기
주변에서 랜덤으로 나타나는 장애물을 피하며 최대한 오래 살아남는 게임입니다.

방향키 또는 WASD키로 움직이고, 쉬프트 키로 대쉬를 사용하면 장애물을 회피할 수 있습니다.

## 소속
- 강릉원주대학교 멀티미디어공학과

## 개발 환경
- 게임엔진: Unity (에디터 버젼: 2022.3.46f1)
- 프로그래밍 언어: C#
- IDE: Visual Studio Code
- 게임서버: 뒤끝SDK (SDK 버젼: 5.14.1)
